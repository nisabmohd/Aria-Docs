## Overview

The `Footer` component is used to render the footer of the website. It typically contains important information such as copyright notices, links to privacy policies, and other relevant details.

### Features:
- Customizable footer links and content.
- Consistent placement at the bottom of the page.
- Supports various layouts for different footer needs.

## Props

| Prop        | Type       | Description                                                         |
| ----------- | ---------- | ------------------------------------------------------------------- |
| `links`     | `array`    | Array of objects defining the links to be displayed in the footer.   |
| `className` | `string`   | Optional additional classes for styling the footer container.        |
