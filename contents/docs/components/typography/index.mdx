## Overview

The `Typography` component is used to provide consistent typography styles across your application. It helps standardize the appearance of text elements like headings, paragraphs, and other inline text.

### Features:
- Supports various text elements such as headings, paragraphs, blockquotes, and more.
- Customizable font sizes, weights, and styles.
- Easily extendable to include custom styles.

## Props

| Prop        | Type       | Description                                                         |
| ----------- | ---------- | ------------------------------------------------------------------- |
| `variant`   | `string`   | The variant of the text, such as "h1", "h2", "p", "blockquote", etc. |
| `className` | `string`   | Optional additional classes for styling the text.                    |
| `children`  | `ReactNode`| The content to be displayed inside the typography component.         |
