## Overview

The `TOC Observer` component works alongside the Table of Contents (`TOC`) to track the user's scroll position and highlight the currently active section in the TOC. This helps users understand which section they are currently viewing as they scroll through the page.

### Features:
- Tracks user's scroll position to highlight active sections.
- Enhances navigation by updating the TOC based on the current section.
- Supports multiple heading levels.

## Props

| Prop        | Type       | Description                                                         |
| ----------- | ---------- | ------------------------------------------------------------------- |
| `headings`  | `array`    | Array of heading elements that the observer will track.              |
| `onActiveChange` | `function` | Callback triggered when the active heading changes.              |
| `className` | `string`   | Optional additional classes for styling the observer container.      |
