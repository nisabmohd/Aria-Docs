## Overview

The `Docs Breadcrumb` component provides navigational breadcrumbs to help users understand their current location within the documentation structure. It is especially useful in larger documentation systems where nested pages are common.

### Features:
- Dynamically generates breadcrumbs based on the current route.
- Supports multi-level navigation.
- Customizable styles and structure.

## Props

| Prop        | Type       | Description                                                         |
| ----------- | ---------- | ------------------------------------------------------------------- |
| `items`     | `array`    | An array of breadcrumb items representing the path structure.        |
| `className` | `string`   | Optional additional classes for styling the breadcrumb container.    |
