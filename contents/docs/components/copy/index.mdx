## Overview

The `Copy` component provides a simple way to allow users to copy text to their clipboard with a single click. This is particularly useful for code snippets, referral links, or any information you want the user to easily copy.

### Features:
- Single-click to copy functionality.
- Customizable button text and styles.
- Easy integration with other components or text content.

## Props

| Prop        | Type       | Description                                                         |
| ----------- | ---------- | ------------------------------------------------------------------- |
| `text`      | `string`   | The text content that will be copied to the clipboard.               |
| `buttonText`| `string`   | The label for the copy button (defaults to "Copy").                  |
| `className` | `string`   | Optional additional classes for styling the button or container.     |

### Use Cases

- **Contact information**: Provide easy-to-copy email addresses or phone numbers.
