## Overview

The `Leftbar` component is a sidebar navigation component that appears on the left side of the page. It is commonly used to provide quick access to various sections or categories within a website or documentation.

### Features:
- Renders a customizable navigation sidebar.
- Supports nested menu items.
- Optional collapsible sections for better organization.

## Props

| Prop        | Type       | Description                                                         |
| ----------- | ---------- | ------------------------------------------------------------------- |
| `items`     | `array`    | An array of menu items to be displayed in the sidebar.               |
| `className` | `string`   | Optional additional classes for styling the sidebar container.       |
