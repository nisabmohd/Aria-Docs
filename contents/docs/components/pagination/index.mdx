## Overview

The `Pagination` component is used to handle paginated data in the application. It allows users to navigate between multiple pages of content, such as lists, tables, or search results.

### Features:
- Supports page numbers, next, and previous controls.
- Customizable number of items per page.
- Easily integrated with lists, tables, or other paginated content.

## Props

| Prop        | Type       | Description                                                         |
| ----------- | ---------- | ------------------------------------------------------------------- |
| `currentPage`| `number`  | The currently active page.                                           |
| `totalPages` | `number`  | The total number of pages available.                                 |
| `onPageChange`| `function`| Callback function called when the page changes.                      |
| `className` | `string`   | Optional additional classes for styling the pagination component.    |
