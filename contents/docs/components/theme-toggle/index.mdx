## Overview

The `Theme Toggle` component provides a simple UI element that allows users to switch between different themes, typically light and dark modes. It integrates with the `ThemeProvider` to update the current theme across the app.

### Features:
- Allows users to toggle between light and dark modes.
- Integrates seamlessly with the `ThemeProvider`.
- Customizable styles and behavior.

## Props

| Prop        | Type       | Description                                                         |
| ----------- | ---------- | ------------------------------------------------------------------- |
| `className` | `string`   | Optional additional classes for styling the toggle button.           |
