## Overview

The `Docs Menu` component is used to render a navigation menu specific to the documentation pages. It helps users easily browse through different sections or categories within the documentation.

### Features:
- Dynamically renders menu items based on the documentation structure.
- Supports nested menu items for hierarchical navigation.
- Customizable design and menu structure.

## Props

| Prop        | Type       | Description                                                         |
| ----------- | ---------- | ------------------------------------------------------------------- |
| `items`     | `array`    | Array of menu items to be rendered in the documentation sidebar.     |
| `className` | `string`   | Optional additional classes for styling the menu container.          |
