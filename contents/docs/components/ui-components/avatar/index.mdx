## Overview

The `Avatar` component is used to display a user’s profile picture or an icon. It’s typically used in user interfaces where personal identification is necessary, such as account settings, comments, or profile pages.

### Features:
- Supports image, icon, or initials.
- Customizable size and shape (circular or square).
- Optionally displays a fallback text if no image is provided.

## Props

| Prop        | Type       | Description                                                         |
| ----------- | ---------- | ------------------------------------------------------------------- |
| `src`       | `string`   | The URL of the image to be displayed.                               |
| `alt`       | `string`   | Alternative text for the image.                                     |
| `size`      | `string`   | Size of the avatar (e.g., "small", "medium", "large").               |
| `shape`     | `string`   | Shape of the avatar ("circle" or "square").                         |
| `className` | `string`   | Optional additional classes for styling the avatar.                 |

## Usage

Here’s an example of how to use the `Avatar` component:

```tsx
import Avatar from '@/components/UI/Avatar';

<Avatar src="/images/user.jpg" alt="User's Profile Picture" size="medium" shape="circle" />
