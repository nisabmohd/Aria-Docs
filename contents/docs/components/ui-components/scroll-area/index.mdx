## Overview

The `Scroll Area` component is used to create containers with customizable scroll behavior. It is commonly used in areas with large amounts of content, such as dropdowns, sidebars, or modals, where scroll functionality is required.

### Features:
- Customizable scroll behavior for vertical or horizontal scrolling.
- Supports scrollable content with smooth scroll options.
- Can be styled to fit into different sections of the application.

## Props

| Prop        | Type       | Description                                                         |
| ----------- | ---------- | ------------------------------------------------------------------- |
| `children`  | `ReactNode`| The content to be displayed inside the scrollable area.              |
| `className` | `string`   | Optional additional classes for styling the scroll area.             |
