## Overview

The `Table` component is used to display data in a structured grid format, with rows and columns. It supports sorting, pagination, and customizable styling, making it perfect for showing tabular data like reports, lists, or any data-heavy interface.

### Features:
- Customizable rows and columns.
- Supports pagination and sorting.
- Easily stylable for different table designs.

## Props

| Prop        | Type       | Description                                                         |
| ----------- | ---------- | ------------------------------------------------------------------- |
| `data`      | `array`    | An array of objects representing the rows of data.                  |
| `columns`   | `array`    | An array of objects representing the column definitions.            |
| `className` | `string`   | Optional additional classes for styling the table.                  |
