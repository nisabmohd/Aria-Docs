## Overview

The `Dropdown Menu` component is used to display a list of selectable items that appear when a user interacts with a button or trigger. Dropdown menus are commonly used for navigation, settings, or actions that need to remain hidden until the user interacts with them.

### Features:
- Displays a dropdown list of selectable options.
- Customizable positioning and animation.
- Supports icons, actions, and nested menu items.

## Props

| Prop        | Type       | Description                                                         |
| ----------- | ---------- | ------------------------------------------------------------------- |
| `items`     | `array`    | An array of menu items, each containing a label and optional action. |
| `onSelect`  | `function` | Callback function triggered when a menu item is selected.            |
| `className` | `string`   | Optional additional classes for styling the dropdown container.      |
