## Overview

The `TOC` component is used to dynamically generate a table of contents based on the headings in the current page. It helps users navigate through large documents by providing a summary of sections and links to them.

### Features:
- Automatically detects headings in the document.
- Generates a clickable table of contents.
- Supports hierarchical headings (H1, H2, H3, etc.).

## Props

| Prop        | Type       | Description                                                         |
| ----------- | ---------- | ------------------------------------------------------------------- |
| `headings`  | `array`    | Array of heading elements (e.g., H2, H3) to be displayed in the TOC. |
| `className` | `string`   | Optional additional classes for styling the TOC container.           |
