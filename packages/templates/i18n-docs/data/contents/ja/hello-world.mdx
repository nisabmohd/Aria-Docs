## 挨拶

これはあなたの**最初のAriadocsページ**です。ここでは、基本的な使い方と構造を簡単に紹介します。

## 概要

Ariadocsは、モダンなReactアプリケーション向けに設計されたドキュメントライブラリです。シンプルで柔軟、そして高速に動作します。

### 特徴

- ⚡ 高速で軽量
- 🎨 カスタマイズしやすい
- 📦 モジュールベース設計
- 🧩 Reactと簡単に統合可能

## はじめに

まずは簡単なコード例を見てみましょう：

```tsx
import java.util.*

console.log("こんにちは世界");
```

## 使い方

Ariadocsを使うには、まずプロジェクトにインストールします：

```bash
npm install ariadocs
```

その後、必要なコンポーネントをインポートして使うことができます。

## セクション例

### 1. セットアップ

プロジェクトの初期設定を行います。

### 2. コンポーネント

Ariadocsが提供するコンポーネントを使ってUIを構築します。

### 3. カスタマイズ

テーマやスタイルを自由にカスタマイズできます。

## 注意点

- 最新のReactバージョンを使用してください
- 設定ファイルの構造に注意してください
