Getting started with Ariadocs in a Next.js **Pages Router** project is simple.

> **Prerequisites**: Complete the [Installation & Configuration](/docs/getting-started/installation) first.

## Add a Docs Page

Use a catch-all route inside `pages/docs/[[...path]].tsx`:

```tsx
// pages/docs/[[...path]].tsx
import { docs } from "@/ariadocs";

export default function Docs({
  MDX,
  frontmatter,
}: {
  MDX: React.ReactNode;
  frontmatter: { title: string };
}) {
  return (
    <section>
      <title>{frontmatter.title}</title>
      {MDX}
    </section>
  );
}

export const getStaticPaths = async () => {
  const rawPaths = await docs.pagePaths();
  return {
    paths: rawPaths.map((str) => ({
      params: { path: str.split("/").slice(1) },
    })),
    fallback: false,
  };
};

export const getStaticProps = async ({ params }) => {
  const slug = (params?.path as string[]).join("/");
  const { MDX, frontmatter } = await docs.parse({ slug });
  return { props: { MDX, frontmatter } };
};
```

## Create Your First Doc

`contents/docs/hello-world.mdx`

```md
---
title: Hello World
---

# Hello World

This is your **first Ariadocs page**.
```

## Run the App

```bash
pnpm dev
```

Open [`http://localhost:3000/docs/hello-world`](http://localhost:3000/docs/hello-world)
